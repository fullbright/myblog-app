# myblog-app

This a cross platform mobile application that uses my blog as a backend.
The technologies used are : 
 - ionic
 - cordova
 - javascript
 - angularjs
 - html / css
 - gravatar
 - wordpress
 - rest api
 - json
 
This application can be compiled into android or iphone operating systems.

# How to run

Here are the steps to run this application in development.

  git clone <this repo's url>
  cd myblog-app
  ionic serve
  
To install the necessary tools, you will need : 

  npm install -g ionic cordova
  npm install -g bower
  bower install gravatar
  
Use sudo if you have permission issues.

# Other libraries

You will need to install android sdk and java sdk.
